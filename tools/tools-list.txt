HOW TO INSTALL ANY TOOL
────────────────────────
  sudo pacman -S <toolname>           Install a single tool
  sudo pacman -S blackarch-<group>    Install an entire category
  sudo pacman -S blackarch            Install ALL 2800+ tools
  sudo mint-tools-setup               Interactive menu installer
  sudo bash tools/install-tools.sh    This repo's install script
